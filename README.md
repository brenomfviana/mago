## Compile and Run

Requirements:

- C++11
- CMake 3.1

To compile the project, go to the root and, through terminal, run the following commands:

```
mkdir build
cd build
cmake ..
```

Then, type the following command to run MAGOS:

```
./MAGOS <width> <height>
```
