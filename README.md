# MAGO

## Description

MAGO is a maze generator (MAze GeneratOr) that uses the Binary Tree approach.

## Build and Run

Dependencies:

- C++17;
- CMake 3.13.

To build the project, go to the root and, through terminal, run the following
commands:

```
mkdir build
cd build
cmake ..
```

Then, type the following command to run MAGO:

```
./MAGO <width> <height>
```
